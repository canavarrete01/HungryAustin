Project Data

Data Sources: Yelp, Yelp API, Recipe Search and Diet API, & REST COUNTRIES API

Description: We took data from the yelp API and the food API and kept the data that was relevant to our attributes. Then, we extracted the cuisine of the food API and assigned the data from Yelp API cuisines from the food API since the cuisines from the food API was more general than the ones for Yelp. Afterwards, we extracted images from flags from the world countries API.
